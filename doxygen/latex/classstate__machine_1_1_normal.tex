\doxysubsection{Detailed Description}
In the normal state the robot reaches random positions. 

When it detects a new colored object it enters the track function. After some times move to the \mbox{\hyperlink{classstate__machine_1_1_sleep}{Sleep}} or \mbox{\hyperlink{classstate__machine_1_1_play}{Play}} behaviour
