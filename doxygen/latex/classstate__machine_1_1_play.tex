\doxysubsection{Detailed Description}
The robot goes to a predefined location and waits to receive a go\+To command which is extracted randomly from the parameter server. 

If the room contained in the command has already been discovered by the robot it moves there. Otherwise it switches to the find state sending as parameter the color of the ball present in the just mentioned room.
