Detailed Description

In the normal state the robot reaches random positions.

When it detects a new colored object it enters the track function. After some times move to the Sleep or Play behaviour

This function is executed everytime the robot detects a color object which was not previously detected.

The robot tries to go closer to the detected object. Once done, the reference position of the room is marked as reached modifying data in the parameter server.
